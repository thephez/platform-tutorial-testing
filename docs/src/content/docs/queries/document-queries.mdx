Dash Platform supports a rich query language for filtering documents stored in data
contracts. You can use comparison operators (equal, greater than, less than, in, starts
with), combine multiple where clauses, specify sort order, and limit results. This
tutorial demonstrates several advanced query patterns against a contract's document store.
