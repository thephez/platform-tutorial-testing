## How It Works

Each tutorial includes an **editable code editor** with a **Run** button. The Dash Platform SDK loads directly in your browser and executes code against the **testnet** — no installation or server required.

1. Browse to a tutorial
2. Read the explanation
3. Click **Run** to execute the example code
4. Edit the code to experiment
5. Click **Reset** to restore the original example
